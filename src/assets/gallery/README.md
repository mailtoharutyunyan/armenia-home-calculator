# Галерея — сюда кладите свои картинки

Просто скопируйте изображения (`.jpg`, `.jpeg`, `.png`, `.webp`) **в эту папку** —
они автоматически появятся в разделе «Галерея» на сайте после пересборки
(`npm run build`) и деплоя (`git push`).

- Никакой настройки не нужно — файлы подхватываются автоматически.
- Имя файла без расширения показывается как подпись под картинкой.
- Рекомендуемый размер: ширина 1200–1600 px, соотношение 4:3 или 16:9.
- Для картинки в шапке (герой) положите файл с именем `hero.jpg` (или `.png`/`.webp`)
  в папку `src/assets/` (на уровень выше этой).

Примеры имён: `фасад-туф.jpg`, `интерьер-зал.jpg`, `план-1-этаж.png`.
