# Калькулятор строительства дома в Армении

Веб-калькулятор, который по параметрам дома считает **объёмы материалов**, **стоимость**
и **сроки** строительства в Армении — отдельно для стадии **«акт / коробка»** и
**«под ключ»**. Проверяет проект по армянским нормам **ՀՀՇՆ**, строит 2D-план,
экспортирует чертёж в **DXF (AutoCAD)** и работает на русском и армянском.

🔗 **Демо:** https://mailtoharutyunyan.github.io/armenia-home-calculator/

> ⚠️ Расчёт ориентировочный и не заменяет проект лицензированного инженера-конструктора.
> Цены — стартовые, требуют актуализации. Разрешение выдают госорганы РА.

## Возможности

- **4 конструктивные системы:** монолитный каркас, несущий туф, газоблок, кирпич.
- **4 типа фундамента** (ленточный, плита, свайный, столбчатый) + подвал.
- **Полная смета:** земляные работы, фундамент, стены, перекрытия, лестница, кровля,
  окна/двери, отделка, инженерные сети — материалы и работа отдельно.
- **Вилка цен поставщиков** (эконом / рыночная / премиум) и цена за м².
- **Стоимость документов и разрешения** в драмах (АПЗ, проект, экспертиза, пошлина и др.).
- **Проверка норм ՀՀՇՆ:** сейсмика 8–9 баллов, этажность, отступы, освещение,
  планировка комнат («помещается ли»).
- **2D-планировка** с мебелью и подписями, экспорт **DXF** и печать/PDF.
- **Таймлайн этапов** со сроками и деньгами на каждый шаг.
- **Сравнение технологий** с плюсами и минусами материалов.
- **Кредитный калькулятор** (первый взнос, ставка, срок → платёж в месяц).
- **Редактор цен** с сохранением в браузере и ссылками на поставщиков.
- Интерфейс **русский + армянский**, валюта **драм / USD**.

## Технологии

React + Vite + TypeScript, Zustand, Tailwind. Всё считается в браузере, без сервера.

## Локальный запуск

```bash
npm install
npm run dev      # режим разработки
npm run test     # тесты движков (Vitest)
npm run build    # продакшн-сборка в dist/
```

## Свои изображения

Скопируйте файлы (`.jpg`, `.png`, `.webp`) в папку **`src/assets/gallery/`** — они
автоматически появятся в разделе «Галерея» после пересборки и деплоя.

## Деплой

Развёртывание на **GitHub Pages** автоматическое через GitHub Actions
(`.github/workflows/deploy.yml`) при каждом `push` в ветку `main`.

## Структура

```
src/
  data/        цены, коэффициенты, регионы, нормы, этапы, разрешение
  model/       типы дома и каталога
  engine/      расчёт объёмов, стоимости, норм, сравнения, 2D-плана, DXF
  ui/          компоненты интерфейса
  i18n/        словари ru / hy
```
